## showcase-car-game
《快上车》休闲小游戏源码，率先升级Creator3.3.0引擎，完全免费，配套完整视频教程！
CocosStore下载地址：https://store.cocos.com/app/detail/2796


## 运行环境
- 引擎版本：Cocos Creator 3.3.0
- 编程语言：TypeScript

## 操作说明
玩家扮演一辆出租车，点按屏幕驱动车在地图上行驶，注意不要碰到其他车辆哦！

将顾客送到指定地点获得金币，解锁更多车型。
